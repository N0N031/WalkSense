# V2.5 — Grille de couverture (spec sauvegardée)

> Statut : retiré de V1.0 le 2026-05-09.
> À réintroduire en V2.5 (M6+) selon référentiel GeoSense V5.0.

## Modèle de données existant
- Session.coverageCells: CoverageCellEntity[]
- Session.lastGridUpdateMs: number
- Session.gridUpdateInterval: number (ms throttle, 500 par défaut)

## Logique d'génération
- generateCoverageFromTrajectory({...}) : génère cellules depuis gpsTrace
- persistCoverageCells(session, cells) : persiste en DB
- Update temps réel pendant session active (throttle 500ms) dans explore.tsx
- Génération finale à la clôture si coverageCells vide

## UI
- Composant GridOverlay (dans SessionMap.*)
- Prop coverageCells passée par SessionMap
- Toggle showGrid

## À noter pour V2.5
- buildCanonical n'incluait PAS les champs grille — choix à reconsidérer
- Limite arbitraire 100 cellules (slice(0, 100)) : à requestionner
- Relier à la "minuterie couverture" V2.5 du référentiel (rouge=sur-exploré,
  vert=optimal, gris=non couvert)
